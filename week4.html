Dirty Data

Underscores.js is a library that provides tools to help cleanup and sort data that can eventually be used to create Interactive visualizations. The underscores.js website has many different examples of all the different ways to sort through and clean data.


Using underscores we can isolate keys, objects or values from arrays by using variables.
